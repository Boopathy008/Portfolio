This folder is reserved for images (e.g. your real profile photo, project
screenshots) if you want to swap out the CSS-based placeholders later.

The site currently uses:
- A styled "MB" initials avatar instead of a profile photo (hero + about section)
- Icon-based gradient banners instead of project screenshots

To use a real photo:
1. Add your photo here, e.g. images/profile.jpg
2. In index.html, replace the two `.avatar-initials` divs with:
   <img src="images/profile.jpg" alt="Mahendra Boopathy">
